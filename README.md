# Запуск:
Запускать утилиту можно даже имея "голую" систему. Но для этого нужно интернет соединение. Wizard-script scrap.cmd поможет в интерактивном режиме установить необходимые компоненты:
  - Python (С python.org или с помощью chocolatey, если он установлен в системе)
  - Pip
  - Virtualenv
  - Виртуальное окружение проекта с необходимымы пакетами
Если все необходимые компоненты имеются, то будет исполнен сам scrapper

# Ограничения:
    1. Wizard работает только ОС семейства Windows ^7
    2. После добавления новых пакетов в requirements.txt необходимо удалить папку venv или вручную запустить "pip install -r requirements.txt"

# Запуск:
    1. Запустите файла scrap.cmd со следующими параметрами (Или "scrap --help"):
        1. от 1 до нескольких url через пробел
        2. --here - вывод новостей в консоль
        3. --adaptive - в случае вывода на экран подгоняет текст под ширину окна
        4. --ticker - построчечный вывод на экран

# Алгоритм:
	1. Нахождение всех параграфов в тексте
	2. От каждого параграфа проиходит подъём по родителям вверх. При проходе узла счётчик у него увеличивается на единицу
	3. Берётся самый "проходной" узел и берется текст из дочерних элементов. Если во время спуска попадается узел, не входящий в [div, span, a, p] отбрасывается
	4. Удаляются бессмыссленные переводы строки
	5. Текст форматируется под необходимую ширину
